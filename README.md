# Conversor de Moedas

## Descrição

Aplicação para conversão de moedas usando Java. 

- Real para Dólar  
- Real para Euro  
- Real para Libra Esterlina  
- Real para Peso Argentino  
- Real para Peso Chileno  
- Dólar para Real  
- Euro para Real  
- Libra Esterlina para Real  
- Peso Argentino para Real  
- Peso Chileno para Real
